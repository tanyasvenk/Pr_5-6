# Internet Shop

## Структура проекта

- **backend-shop** – сервер для отображения каталога товаров (порт 3000)
- **backend-admin** – сервер для админ-панели с API для управления товарами (порт 8080)
- **frontend** – простая HTML-страница для панели админа
